# 控制器 HealthController : 健康状态控制器
- 可配置属性
    - beforeActionCallback ： beforeAction 前需要执行的回调函数
    - afterActionCallback ： afterAction 前需要执行的回调函数

##  使用方法
### 1. 方法一 ： 文件配置

```php
'controllerMap' => [
    'health' => [
        'class'                => HealthController::class,
        'beforeActionCallback' => function (\yii\base\Action $action) {
            var_dump('property - beforeActionCallback : ' . $action->getUniqueId());
            return true;
        },
        'afterActionCallback'  => function (\yii\base\Action $action, $result) {
            var_dump('property - afterActionCallback : ' . $action->getUniqueId());
            return $result;
        }
    ],
],
```

### 2. 方法二 ： 控制器继承

```php

namespace app\controllers;


use yii\base\Action;
use yii\helpers\ArrayHelper;

class HealthController extends \YiiHelper\controllers\HealthController
{
    public function behaviors()
    {
        return ArrayHelper::merge(parent::behaviors(), [
            'actionCallback' => [
                // 指定 beforeAction 回调
                'beforeActionCallback' => [
                    [$this, 'beforeIndex'],
                ],
                // 指定 afterAction 回调
                'afterActionCallback'  => [$this, 'afterIndex'],
            ]
        ]);
    }


    public function beforeIndex(Action $action)
    {
        var_dump('filter - beforeActionCallback : ' . $action->getUniqueId());
        return true;
    }

    public function afterIndex(Action $action, $result)
    {
        var_dump('filter - afterActionCallback : ' . $action->getUniqueId());
        return $result;
    }

    public function actionIndex()
    {
        return parent::actionIndex(); // TODO: Change the autogenerated stub
    }
}
```

### 3. 方法三 ： 结合方法一和方法二
